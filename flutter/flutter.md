# webformatURLとは
Pixabay API(Pixabay API は、世界中のクリエイターが投稿した高品質な画像、動画、音楽を無料で提供しているサイト)から返ってくる情報のひとつで、「Webでの表示に適した中サイズ（最大幅640ピクセル）の画像のURL」のこと

# class
classはクッキーの型みたいなもの。
自分でこういう型のデータがあるよ、と定義できる

# インスタンス
実態。クッキーの型を使って型抜きしたクッキーそのもの。

# コンストラクタ
インスタンスを作るための関数

# list
複数のデータを順番に並べて管理するための箱

# map
ラベル（名前）で中身を取り出す、情報のセット
例）
'{
  'likes': 100,
  'url': 'https://...',
}'

# factry
mapを引用したコンストラクタが作れる

# flutter pub add
パッケージを追加できる。
複数のパッケージをスペースをあけて書くことで複数同時に追加可能。

# flutter pub add -d
このオプションを追加すると dev_dependencies にパッケージが追加される。普通の dependencies と違い dev_dependencies に追加されたパッケージは lib フォルダの中で使うことはできない。その代わり依存関係を少なくすることができるため pub コマンドの実行速度が早くなる。
開発中のみ使用可能で、リリースされる本番アプリには使えない。

# 名前付き引数
引数を{}で囲むだけでok。
引数の順番を覚えなくていい。何番目の引数は色だっけ？となららなくていい。必要なものだけ指定できる。かつ読みやすくなる。

# required
名前付き引数を定義するときに、この値は絶対に省略させないと強制するためのキーワード。名前付き引数は本来あってもなくてもいいという性質を持つので、その値がないと動かないようにすることができる。

# @Entity
ObjectBoxなどのデータベースを使う際に、このクラスをデータベースのテーブル（保存する箱）として使うことを宣言するための印（アノテーション）。
'int id = 0'のidが必須。

# ??
'変数 ?? 代入したい値' 左側がnullじゃない時、そのまま左の値を使う。nullの時は右側の値を使う。

# getAll()
その箱（Box）の中に保存されている全てのデータを、リスト（List）形式で一気に取り出す」 という命令

# body
メインコンテンツ（中身）を入れるための場所で、ほとんどの場合、画面の土台となる Scaffold という Widget の中で使われる。
