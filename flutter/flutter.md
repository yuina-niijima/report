# webformatURLとは
Pixabay API(Pixabay API は、世界中のクリエイターが投稿した高品質な画像、動画、音楽を無料で提供しているサイト)から返ってくる情報のひとつで、「Webでの表示に適した中サイズ（最大幅640ピクセル）の画像のURL」のこと

# class

# インスタンス

# list
